# Project_Clickmart

Clickmart is an online grocery e-commerce system backend built using Django Rest Framework (DRF).

## Features
- User authentication
- Product management
- Cart system
- Order handling
- APIs for frontend integration

## Tech Stack
- Python
- Django
- Django Rest Framework
- PostgreSQL
